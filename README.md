# QR 코드 생성기

## 개요
- **목표**: URL → QR 코드 이미지 생성 → 다운로드
- **수익 모델**:
  - 무료: 기본 QR 코드
  - 유료: 커스텀 색상, 로고 추가, 일괄 생성
- **자동화**: 완전 자동 (결제 → 생성 → 다운로드)

## 기술 스택
- **프론트엔드**: Next.js 15 + TypeScript + Tailwind CSS
- **QR 생성**: qrcode 라이브러리
- **결제**: Stripe (나중에)
- **호스팅**: Vercel (무료)

## 기능
- ✅ URL 입력 → QR 코드 생성
- ✅ QR 코드 다운로드 (PNG)
- ⏳ 커스텀 색상 (유료)
- ⏳ 로고 추가 (유료)
- ⏳ 일괄 생성 (유료)
